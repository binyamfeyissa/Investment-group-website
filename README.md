## Installation

### Clone the repository:

```bash
git clone https://github.com/d3adrabbit/origami.git
```

### Navigate to the project directory:

```bash
cd origami
```

### Install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

## Usage

### Running the Development Server

To start the development server, run:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
